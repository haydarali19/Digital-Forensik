Writeup Digital Forensik

Drone di Tengah Salju

Di challenge ini kita diminta membaca jejak penerbangan drone, memahami alur misinya, lalu menarik informasi penting dari log mentah sampai ketemu titik yang paling relevan untuk menjawab soal.

Kategori: Forensik Drone Format: HTML/CSS Statis Gaya: Writeup Modern

Ringkasan Singkat

Rute misi, pesan crash, dan data GPS saling menguatkan.

Alurnya saya buat seperti writeup referensi: mulai dari melihat gambaran besar misi, masuk ke log, lalu mengerucut ke bukti inti secara pelan-pelan sampai jawabannya benar-benar terasa masuk akal.

Mulai investigasi

Gambaran Challenge

Skenario

Sebuah disk yang berhasil dipulihkan membuka arah penyelidikan. Beberapa saat kemudian, sebuah drone hitam tanpa tanda ditemukan setengah tertimbun salju di Budapest. Dari lab file yang diberikan, kita diminta merekonstruksi rute, membaca perilaku penerbangan, dan menunjukkan apa yang terjadi sebelum drone menghantam tanah.

Tools

Alat yang dipakai

  • QGroundControl untuk membaca jalur misi
  • mavlogdump.py untuk membongkar isi log
  • Filter MSG dan GPS untuk bukti inti

Tinjauan Awal

Lab file yang dipakai

Ada dua file utama di challenge ini. File AOTR_Winter_Blackout.plan berisi rencana misi, waypoint, dan command yang dijalankan drone. Sementara log.bin berisi kejadian aktual selama penerbangan. Jadi urutan analisis paling enak memang dimulai dari file plan, baru setelah itu pindah ke log.
